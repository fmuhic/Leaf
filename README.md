## Leaf

Leaf is physics engine for 2D games.

#### Dependenices

```sh
# OpenGL
sudo apt install mesa-utils libglu1-mesa-dev freeglut3-dev mesa-common-dev
sudo apt install libao-dev libmpg123-dev

```
#### Build Using CMAKE
```sh
mkdir build
cd build
cmake ../
# On Windows use nmake
nmake .
# On Linux use GNU make
make .
```
